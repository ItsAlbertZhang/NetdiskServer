# 配置文件内容

> C 语言造轮子太麻烦, 不想写标准 json 格式了, 就这么凑合着用吧.

**注意! 所有 .config 配置文件需要在最后留空一行! 否则 fgets 函数将无法正确获取配置文件.**

### mysql.config

第一行: 服务器 ip.

第二行: MySQL 用户名.

第三行: 要使用的数据库(database)名.

### pthread.config

第一行: 子线程数量.

第二行: 最大等待子线程服务的连接数. 一个建议的数值是子线程数量的 2 倍.

### tcp.config

第一行: 服务端所使用的 ip 地址.

第二行: 服务端所使用的端口号.

第三行: 服务端签名. 这个签名是针对将日志写入 MySQL 数据库时在 server_sign 字段中填入的内容而言的.
